Códigos de Retorno HTTP

200 (OK) - GET requests.

201 (Created) - POST e PUT salvar novos cliente

204 (No Content) - for PUT, e DELETE

400 (BAD_REQUEST) Outro Erros

Exemplos de Consultas via GET

Buscar clientes por pagina a partir da pagina 5:

page=Página Requerida

size=Quantidade de itens por pagina(default = 10)

http://localhost:8080/api/clientes?page=1&size=5

Buscar Paginada com Links (primeiro, próximo, anteriro e última

http://localhost:8080/api/clientes?page=1&size=3

Busca por nome iniciando com parametro[nome]:

http://localhost:8080/api/clientes/search/noneStartsWith?nome=G

Buscar Cliente por Identificador::

http://localhost:8080/api/clientes/1

Buscar Cliente pro CPFCNPJ

http://localhost:8080/api/clientes/search/documento?cpfcnpj=7894900011517

Exemplos de Consultas via POST, DELETE

Excluir Cliente (Http: DELETE)

http://localhost:8080/api/clientes/1(id)

Salvar ou Atualizar Cliente (POST). Definir o content type do body como JSON no cliente rest

http://localhost:8080/api/clientes
